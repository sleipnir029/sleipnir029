## Setup Instructions

1. **Replace `YOUR_USERNAME`** with your GitHub username in all URLs above
2. **Enable GitHub Pages** for this repository:
   - Go to Settings → Pages
   - Source: Deploy from a branch
   - Branch: main → / (root)
   - Save
3. **Upload `profile-animation.html`** to your repository root
4. **Update the animation link** (line 4) with: `https://YOUR_USERNAME.github.io/YOUR_USERNAME/profile-animation.html`
5. Your animated profile will be live at that URL!
