# ASCII Profile Draft

This repo is a minimal GitHub-profile animation setup.

## Build

```bash
python src/build_animation.py
```

It generates:
- `assets/current.gif`
- `assets/current.webp`
- `assets/poster.png`
- editable text frames in `src/frames/`

## Edit

- Change the base ASCII art or motion logic in `src/build_animation.py`
- Re-run the build command
- Commit updated assets
